## ðŸ’¡ TIPS FOR SUCCESS

### Customization is Key
- **Never** send generic cover letters
- **Always** research the gallery/institution
- **Mention** specific exhibitions or programs they've done
- **Explain** why you're a good fit for them specifically

### Professional Presentation
- Use high-quality paper for printed versions
- Ensure consistent formatting across documents
- Proofread everything multiple times
- Have someone else review before sending

### Digital Best Practices
- PDF format for official submissions
- Name files clearly: `Noble_Danso_Portfolio_2025.pdf`
- Keep file sizes reasonable (<5MB for email)
- Include contact info on every page

### Follow-Up Strategy
- Wait 2 weeks before initial follow-up
- Send brief, polite inquiry
- Offer additional materials
- Thank them for consideration
